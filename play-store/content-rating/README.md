# Content Rating (IARC Questionnaire)

Completed inside Play Console (IARC questionnaire), not uploaded as a file —
this doc tracks the answers so they're consistent and auditable.

## Expected answers for Amoura
- **Violence**: None
- **Sexual content**: Romantic/suggestive themes present (Spicy Mode) —
  answer honestly as "Sexual content — suggestive themes," NOT "no sexual
  content." Under-declaring here risks takedown after the fact.
- **User-generated/AI-generated content**: Yes — chat and generated images are
  AI-generated per user prompts; disclose the app includes AI-generated
  content and a way to report objectionable output (the in-app feedback tool).
- **Target age group**: 18+
- **Data collection**: match whatever the Data Safety form says (see
  `../data-safety/`)
- **Ads**: answer "Yes, this app shows ads" (banner, interstitial, rewarded via
  AdMob). Whether ads are personalized depends on the UMP consent outcome per
  user/region — do not claim "no ads" or "non-personalized only" globally.
- **In-app purchases**: answer "Yes" (single bundled subscription via Google
  Play Billing that unlocks Spicy Mode and removes ads)

## Result
Expect a **Mature 17+** (or regional equivalent) rating. Do not attempt to
game this down to Teen/Everyone — mismatched rating vs. actual content is a
common cause of Play Store suspension for this app category.
